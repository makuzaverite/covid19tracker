#### How to use it on your computer

```git
  git clone https://github.com/makuzaverite/covid19tracker.git
```

```git
  cd covid19tracker
```

> in terminal do

```git
   npm install
```

```git
   npm start
```

> App should be opened in browser on post of 3000
